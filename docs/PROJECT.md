---

## 1. Overview

This is rebel Financial's (an RIA / Registered Investment Adviser) **client-facing digital
asset market education dashboard**. It presents the structure of the crypto market —
total market cap, leader dominance, sector distribution, volatility, single-asset quotes —
as a clean web page, helping advisors explain to clients "what this market looks like,
what it is made of, and who leads it."

It is **not** a trading tool and offers no investment advice. Its purpose is
"market-structure education + research transparency."

---

## 2. Purpose & background

- **Educate clients.** Most clients lack a structured view of the crypto market. The
  dashboard uses a market-cap bubble map, sector shares, and leader trends to make the
  abstract market concrete.
- **Show research transparency.** Starting from 683 Anchorage-supported assets, a set of
  screening rules narrows the field to a focused research set — the whole funnel is visible
  and traceable, demonstrating the rigor of the advisory research process.
- **Build trust.** Real data + source attribution + a compliance disclaimer meet the
  credibility and compliance bar expected of RIA client-facing material.

## 4. Architecture

```
Browser (index.html — modular static SPA)
  │  ├─ Chart.js              → share / market-cap / volatility charts
  │  ├─ TradingView embed     → single-asset live price chart
  │  └─ src/data/overview-assets.js → 683-asset universe metadata
  │
  │  poll every 60s             fetch every ~6h (cache)
  ▼                             ▼
/api/quotes  ───────┐    /api/history ───────┐
(Vercel function)   │    (Vercel function)   │
  CMC_API_KEY server-side │   CMC_API_KEY server-side │
  edge cache 300s     │      edge cache 6h        │
  ▼                   │      ▼                    │
CoinMarketCap         │    CoinMarketCap (current cap / supply)
(listings/latest)     │    DeFiLlama (historical prices)
                      └─ the API key never reaches the browser
```

- **Frontend.** Static, build-free site (`index.html` + `styles/*.css` + ES modules in `src/`),
  Chart.js + TradingView, deployed on Vercel. See [architecture.md](architecture.md).
- **Backend (web).** Two Vercel serverless functions act as proxies; the API key lives in an
  environment variable (`CMC_API_KEY`) and is never exposed to the client.
- **Caching.** Live quotes are edge-cached for 300s (60s polling still hits cache, keeping
  the free tier safe); historical data is cached for 6h.
- **Degradation.** If any endpoint fails, the page falls back to snapshot data and shows a
  status indicator (LIVE green / SNAPSHOT red).

## 5. Features

### Three main views (tabs)
- **Overview.** A 683-asset market-cap bubble map (the Anchorage-supported universe), with a
  toggle to narrow to the 54 "kept" assets and an explanation of the screening rules.
- **Dashboard.** The core market board:
  - Top KPI bar: total market cap / BTC dominance / ETH share / ex-BTC·ETH cap / sector count
  - Asset ticker strip (live price + 24h change)
  - Sector market-share donut + TradingView single-asset chart
  - Market share (stacked area) + annualized volatility
  - Top assets table (price / 24h / market cap / share / sector)
  - Total market-cap trend (30d / 1y)
- **Sectors.** 13 sector cards → sector detail table (sortable) → single-asset detail (with TradingView).

### Data & credibility
- **Live quotes** from CoinMarketCap, 60s refresh, LIVE status light.
- **Real history**: total market cap / dominance / volatility all from real data
  (CMC + DeFiLlama); no hardcoded or synthetic series.
- **Source attribution** on each historical chart.
- **Compliance disclaimer**: "For educational purposes only — not investment advice."

## 6. Data sources

| Data | Source | Endpoint | Frequency / cache |
|---|---|---|---|
| Live quotes, market cap, 24h change | CoinMarketCap | `/v1/cryptocurrency/listings/latest` | 60s poll / 300s cache |
| Historical prices | DeFiLlama | `coins.llama.fi/chart` | 6h cache |
| Single-asset price chart | TradingView | embedded widget | live |
| Universe metadata | local `src/data/overview-assets.js` | — | static (generated by the pipeline) |

> Note: the current CMC plan does not expose the historical endpoint, so historical data
> comes from DeFiLlama; market-cap history is reconstructed as `current_cap × historical_price / current_price`.

## 7. Deployment

- **Hosting**: Vercel (auto-deploys from the GitHub `main` branch).
- **Env var**: `CMC_API_KEY` (configured in Vercel, Sensitive, Production + Preview).
- **Published source**: `index.html` + `styles/` + `src/` + `api/` (zero-config — `index.html`
  and `/api` stay at the repo root).
- **Secret safety**: a whitelist `.gitignore` publishes only the web app + docs; the Python
  research pipeline and any credential files are never committed.
